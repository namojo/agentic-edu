Cursor 치트시트

Rules · Commands · Skills · Subagents · MCP — 각 기능의 "어디에 뭘 두느냐"를 한 표로. 2026-04-23 기준 cursor.com/docs 근거.

기능 프로젝트 경로 전역(User) · Windows 전역(User) · macOS (보조) 메모
Rules .cursor\rules\*.mdc %USERPROFILE%\.cursor\rules\ ~/.cursor/rules/ 항상 프롬프트에 붙음. 구 "Rules for AI"는 레거시, 현 공식명은 "Rules".
Commands .cursor\commands\ %USERPROFILE%\.cursor\commands\ ~/.cursor/commands/ Cursor 3.x부터 Skills로 흡수 추세. /migrate-to-skills 제공.
Skills .cursor\skills\<name>\SKILL.md %USERPROFILE%\.cursor\skills\<name>\SKILL.md ~/.cursor/skills/<name>/SKILL.md frontmatter의 description이 트리거. 자동 로드 또는 /skill-name.
Subagents .cursor\agents\<name>.md %USERPROFILE%\.cursor\agents\<name>.md ~/.cursor/agents/<name>.md 공식 /create-subagent 명령 없음. 수동 파일 또는 자연어로 생성 요청.
MCP .cursor\mcp.json %USERPROFILE%\.cursor\mcp.json ~/.cursor/mcp.json 동명 서버는 프로젝트가 우선. Settings → "Add new global MCP server"로 편집기 오픈.
하이브리드 환경 리마인드

Cursor Free 플랜은 BYOK로 Gemini 모델을 수동 선택할 수 없습니다. 본 강의는 Cursor 내장 LLM(대화·편집)과 별도 터미널의 Gemini API 호출(Python)로 분리한 하이브리드를 사용합니다.

공유 체크리스트
  • Skill/Agent 디렉토리를 git에 커밋 (팀 공유 표준)
  • README에 설치 경로 명시 — Windows: %USERPROFILE%\.cursor\..., macOS: ~/.cursor/...
  • Cursor Directory는 "복붙 방식"이 공식 설치 경로

공식 MCP 서버 5종 카탈로그

레포: modelcontextprotocol/servers. mcp.json의 command: "npx"로 바로 실행되는 기본 5종.

filesystem

파일 읽기/쓰기/검색, 허용 디렉토리 제어. 본 강의 실습 A의 핵심 서버.

npx -y @modelcontextprotocol/server-filesystem <dir>
git

Git 리포 읽기/검색/조작. 브랜치·커밋 로그 조회 등 개발 워크플로 시연용.

npx -y @modelcontextprotocol/server-git
memory

지식 그래프 기반 영구 메모리. "에이전트가 기억하게 하기" 컨셉.

npx -y @modelcontextprotocol/server-memory
fetch

웹 페이지 가져오기 + markdown 변환. 실습 B 팩트체크 단계에서 유용.

npx -y @modelcontextprotocol/server-fetch
time

시간/시간대 변환. 가장 단순한 MCP 서버로 첫 튜토리얼에 최적.

npx -y @modelcontextprotocol/server-time
추가 참고

sequential-thinking(단계 분해), everything(학습용 통합 샘플). 전체 카탈로그는 modelcontextprotocol.io/examples.

Gemini 무료 티어 한도

2026-04 기준. Google AI Studio 대시보드에서 계정별 실제 수치 확인 권장. API Key 발급 페이지.

모델 RPM (분당) RPD (일일) 추천 용도
gemini-2.5-pro 5 100 최고 성능, 일일 한도 가장 타이트
gemini-2.5-flash 10 250 범용 기본 추천 (본 강의 기본값)
gemini-2.5-flash-lite 15 1,000 경량, 교실 단체 실행 시 권장
20명 교실에서 동시에 실습 B을 돌리면 gemini-2.5-flash 250 RPD는 빠르게 소진될 수 있습니다. 이럴 땐 (1) gemini-2.5-flash-lite로 전환 (2) 더미 데이터 모드로 I/O만 시연 (3) 순서제 실행 중 택1. Google은 2025-12-07부터 무료 티어를 50~80% 축소했으니 숫자는 "2026-04 기준"으로 명시하세요.

심화 과제 C1 ~ C3

수업에서는 아이디어만 제시합니다. 실습 B를 완료한 뒤 직접 확장해 보세요.

C1 · 병렬 확장

일본 뉴스 Agent 추가

C2 · 크로스 체크

Playwright로 원문 URL 검증

C3 · 배달 자동화

Slack 웹훅 자동 배달

orchestrator 종료 훅에 incoming webhook POST 추가. OS 스케줄러 또는 GitHub Actions로 매일 오전 9시 자동 실행.
