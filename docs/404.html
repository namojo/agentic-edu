Why Now

왜 지금 Agentic AI를
직접 만들어야 하는가

"Skill · MCP · Agent · Multi-Agent" — 이 네 조각을 이해하는 순간, 업무 자동화는 더 이상 개발팀만의 영역이 아닙니다. 본 과정은 개념·실전·환경 세 축을 하루에 동시에 잡습니다.

개념: 4단 키워드 지도

Skill(매뉴얼) → MCP(도구 플러그) → Agent(전담 비서) → Multi-Agent(팀). 비유와 그림으로 한 장에 정리합니다.

실전: 두 개의 산출물

실습 A는 "다운로드 폴더 정리해줘" 한 마디로 Skill + MCP가 돌아가는 경험. 캡스톤 B는 키워드 하나로 4개 에이전트가 한·미 뉴스를 취합·검증하는 미니 뉴스룸입니다.

환경: Plan C 하이브리드

Cursor Free + 내장 LLM으로 IDE 경험을, Gemini 무료 API는 별도 터미널/Python으로 호출합니다. BYOK 가정 없음 · 라이선스 리스크 없음.

파일 관리 MCP + Skill

  • SKILL.md 1개 (트리거·절차·완료기준)
  • 로컬 MCP 서버 1개 (list/move/find_duplicates)
  • Cursor 등록 스크린샷 + 실행 로그

B

한/미 뉴스 멀티에이전트

4개 Agent + 파일 MCP 재사용. 키워드 1개 → 병렬 수집 → 팩트체크 → 통합 리포트. 본 과정의 캡스톤.

  • kr-news · us-news · fact-checker · orchestrator
  • 공용 `_workspace/*.md` 중간 산출물
  • '검증 경고' 섹션 자동 생성
캡스톤 B 자세히

자주 묻는 질문

수강에 비용이 드나요?
기본은 모두 무료입니다. Cursor Free 플랜과 Gemini 무료 API(gemini-2.5-flash 또는 flash-lite)로 전 과정을 완주할 수 있도록 설계됐습니다.
사전 지식이 필요한가요?
업무용 문서 편집이 가능하고 터미널 명령어를 안내에 따라 복사·붙여넣기할 수 있으면 충분합니다. 코드 경험은 필요 없습니다.
얼마나 걸리나요?
총 4시간(240분) 기준입니다. 모듈 합은 약 280분이며 휴식·Q&A 시간으로 흡수합니다. 모듈 03·04·05는 각 5분 버퍼가 내장돼 있습니다.
Cursor 무료 플랜으로 충분한가요?
충분합니다. 다만 Free 플랜은 Auto 모드만 가능하고 Gemini BYOK로 모델을 직접 고를 수 없습니다. 그래서 Plan C(하이브리드) — Cursor 내장 LLM + 별도 터미널 Gemini 호출 — 로 진행합니다.
Gemini 한도가 부족하면?
gemini-2.5-flash는 2026-04 기준 10 RPM / 250 RPD. 수업 인원 20명이 동시에 캡스톤을 돌리면 빠듯할 수 있어 flash-lite(15 RPM / 1000 RPD)로 전환하거나 더미 데이터 모드를 제공합니다.
실습을 못 따라가면 어떻게 하나요?
각 모듈 말미에 "체크포인트"와 완성 상태의 공유 템플릿이 제공돼 중간부터 합류 가능합니다. 또한 실습 A·B는 독립적으로 구성돼, B만 해도 무방합니다.
